ac-export-excel

$ ynpm install ac-export-excel --save

引入

import ExportExcel from 'ac-export-excel';

描述

当前导出组件，不依赖任何后台接口，仅是前端的导出下载功能。使用new ActiveXObject("Excel.Application") 功能 实现下载功能
